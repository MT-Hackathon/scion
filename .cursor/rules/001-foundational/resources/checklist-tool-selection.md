# Checklist: Tool Selection

Verify before creating scripts or choosing tools.

---

## Pre-Script Creation

- [ ] Have I checked rule-owned `scripts/` folders for existing workflows?
- [ ] Have I listed the relevant rule scripts?
- [ ] Is this functionality unique and not a duplicate?

## Tool Selection

- [ ] Could this be scripted to save tokens or reduce attention attenuation?
- [ ] Am I using progressive context discovery?
- [ ] Have I included script-based verification in my plan?

## Prohibited Actions

- Duplicating existing script functionality
- Reading, catting, grepping, or sourcing `.env` files directly
