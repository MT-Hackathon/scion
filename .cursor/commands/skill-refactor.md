# skill-refactor

Review our skills for compliance to the scaffold rule and the skills skill. Consider the relevance and accuracy of what is in the skills. Is is consistent with other skills? with the current project (if applicable)? with the current language (if applicable)? Should any of the skills in those sections be generalized to apply more broadly? do we need to consolidate skills or split them apart? Are the anchors all up to date with good descriptions?

Consider all of these and produce a report. If the report indicates changes are needed, it is likely a plan to make changes will be next.
