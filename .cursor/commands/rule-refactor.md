# rule-refactor

Review our rules for compliance to the scaffold rule and the rule skill. Consider the relevance and accuracy of what is in the rules. Is is consistent with other rules? with the current project (if in 200-399 range)? with the current language (if in 100-199 range)? Should any of the rules in those sections be generalized to apply more broadly? do we need to consolidate rules or split them apart? Are the anchors all up to date with good descriptions?

Consider all of these and produce a report. If the report indicates changes are needed, it is likely a plan to make changes will be next.
